Day 0-30

Foundation

The initial 30 days are dedicated to building a rock-solid engineering mindset. We immerse our talent in core platform technologies and cloud fundamentals.

Phase Description

During the foundation phase, fresh engineers are introduced to the core tools and practices that drive modern enterprise platforms. The goal is to quickly bring them up to speed on cloud environments, coding standards, and collaborative methodologies. Emphasis is placed on practical, hands-on learning rather than purely theoretical constructs.

Key Technologies

  • Google Cloud Platform: Infrastructure basics, IAM, and networking.
  • LangChain: Fundamentals of LLM orchestration and API integration.
  • Python & Git: Industry-standard version control and scripting.
  • Data Processing: Initial setups for Big Data processing pipelines.

Learning Journey

  • Data training pipeline architecture
  • Engineering mindset workshops
  • Soft skills and consulting etiquette
  • Continuous integration principles
